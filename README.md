# 📈 Stock Analytics Dashboard

> Real-time stock market analytics, portfolio tracking, and news sentiment — all in one sleek dashboard.

A powerful **Python Dash** web app that lets you analyze any US stock with live candlestick charts,
RSI/MACD technical indicators, portfolio P&L tracking, and real-time news sentiment.

🎬 [Watch Demo Video](https://github.com/Bhanu17291/stock-analytics/issues/1#issue-4069633507)

---

## ✨ Features

- 📊 **Candlestick Charts** — Interactive price charts with multiple time periods
- 📉 **Technical Indicators** — RSI and MACD charts for in-depth analysis
- 💼 **Portfolio Tracker** — Add/remove stocks, track invested value and P&L
- 📰 **News & Sentiment** — Latest news feed for any stock
- ⚡ **Auto Refresh** — Data updates every 5 minutes aut
